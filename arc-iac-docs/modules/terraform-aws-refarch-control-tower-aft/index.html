    terraform-aws-refarch-control-tower-aft¶

    Overview¶

    SourceFuse AWS Reference Architecture (ARC) Terraform module for managing Control Tower Account Factory Terraform.

    Usage¶

    To see a full example, check out the main.tf file in the example folder.

    1
2
3
    module "aft" {
  source = "git::https://github.com/sourcefuse/terraform-aws-refarch-control-tower-aft.git"
}

    Requirements¶

    Name Version
    terraform >= 1.3
    aws ~> 4.0

    No providers.

    Versioning¶

    This project uses a .version file at the root of the repo which the pipeline reads from and does a git tag.

    When you intend to commit to main, you will need to increment this version. Once the project is merged, the pipeline will kick off and tag the latest git commit.
